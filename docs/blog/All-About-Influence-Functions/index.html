Influence functions

Note that this material is mainly a summary of the math in Ben Jann's notes compiled together with additional writing by myself.

Setup

Let zn={zi}i=1n{\bf z}_n = \{z_i\}_{i=1}^nzn​={zi​}i=1n​ be a random sample of data that comes from some true underlying distribution FFF. We take this data and compute some estimator with it: θ^(zn)\hat{\theta}({\bf z}_n)θ^(zn​) (scalar or vector). Note that this is a function of the sample we observe, zn{\bf z}_nzn​. Some examples:

  1. Sample Mean: ziz_izi​ is a scalar and θ^(zn)\hat{\theta}({\bf z}_n)θ^(zn​) the sample mean of the data.
μ=EF[zi]\mu = \mathbb{E}_F\left[ z_i \right]μ=EF​[zi​]
  1. Regression: {zi≡(xi,yi)}i=1n\{z_i \equiv (x_i, y_i)\}_{i=1}^n{zi​≡(xi​,yi​)}i=1n​ be a random sample of data, where xix_ixi​ is a 1×k1\times k1×k vector of covariates and yiy_iyi​ is a scalar outcome. θ^(zn)\hat{\theta}({\bf z}_n)θ^(zn​) would be the OLS coefficients β\betaβ.
β=argminβEF[(y−Xβ)′(y−Xβ)] \beta = argmin_{\beta} \mathbb{E}_F\left[ (y-X\beta)'(y-X\beta) \right]β=argminβ​EF​[(y−Xβ)′(y−Xβ)]
  1. Treatment Effect: {zi≡(yi,Di)}i=1n\{z_i \equiv (y_i, D_i)\}_{i=1}^n{zi​≡(yi​,Di​)}i=1n​ where yiy_iyi​ is the outcome and DiD_iDi​ is the treatment indicator. θ^(zn)\hat{\theta}({\bf z}_n)θ^(zn​) would be the treatment effect on the treated. The estimate would be the sample analogue of
τ=EF[yi∣Di=1]−EF[yi∣Di=0]\tau = \mathbb{E}_F\left[ y_i \vert D_i = 1\right] - \mathbb{E}_F\left[ y_i \vert D_i = 0\right]τ=EF​[yi​∣Di​=1]−EF​[yi​∣Di​=0]

The influence function

First, we define a contaminated distribution function, Fϵ(zi)F_\epsilon(z_i)Fϵ​(zi​), as:

Fϵ(zi)=(1−ϵ)F+ϵδziF_\epsilon(z_i) = (1-\epsilon)F + \epsilon\delta_{z_i}Fϵ​(zi​)=(1−ϵ)F+ϵδzi​​

where δzi\delta_{z_i}δzi​​ is the probability measure which assigns probability 1 to zi=(xi,yi)z_i = (x_i, y_i)zi​=(xi​,yi​) and 0 to all other elements. In effect, Fϵ(zi)F_\epsilon(z_i)Fϵ​(zi​) makes data point zi=(xi,yi)z_i = (x_i, y_i)zi​=(xi​,yi​) slightly more likely in the population. To make clear, if ϵ=0.5\epsilon = 0.5ϵ=0.5 that means with (at least) probability 1/2 you observe ziz_izi​ given a random draw from Fϵ(zi)F_\epsilon(z_i)Fϵ​(zi​).

Our goal is to see what happens to our estimator when we increase the probability of seeing ziz_izi​ in the population. This gives us a sense of how ziz_izi​ influences the sample distribution of the estimator θ^(zn)\hat{\theta}({\bf z}_n)θ^(zn​).

To build intuition, let's think about outliers in regressions. If one observation, ziz_izi​, is a high-leverage outlier, then intuitively its presence has a lot of influence on the regression coefficients. Formally, the influence function asks if you make this ziz_izi​ slightly more likely, how much does it move the estimated coefficients, β^\hat{\beta}β^​.

To formalize this, we will use what's called a "Gateaux derivative" which is just the fancy version of a derivative. The influence function of θ^\hat{\theta}θ^ at FFF, IFθ^,F(zi)IF_{\hat{\theta}, F}(z_i)IFθ^,F​(zi​) is defined as:

IFθ^,F(zi)=lim⁡ϵ→0θ(Fϵ(x))−θ(F)ϵ IF_{\hat{\theta}, F}(z_i) = \lim_{\epsilon \to 0} \frac{\theta(F_\epsilon(x)) - \theta(F)}{\epsilon}IFθ^,F​(zi​)=ϵ→0lim​ϵθ(Fϵ​(x))−θ(F)​

This is a slight change of notation as we are now not specifying a particular sample zn{\bf z}_nzn​, but instead the distribution it is drawn from. The influence function is worked out based on the actual population moments that give rise to the sample estimates. This will hopefully make more sense when we work out some examples below.

Influence function and Variance of Estimator

What's helpful about knowing the influence function is that we can think of our sample estimator as being equal to the true value (so long as we have unbiasedness) plus nnn disturbances of the distribution with weight ϵ=1n\epsilon = \frac{1}{n}ϵ=n1​ for each. Each disturbance causes the true estimate to be influenced (or moved) by approximately 1n∗IFθ^,F(zi)\frac{1}{n} * IF_{\hat{\theta}, F}(z_i)n1​∗IFθ^,F​(zi​) (derivative times a change in x). Since we are extrapolating the derivative by a distance of 1n\frac{1}{n}n1​, this gives rise to higher order terms from the Taylor expansion:

θ^(zn)=θ0⏟unbiasedness+∑i=1n1n∗IFθ^,F(zi)⏟approx. influence of zi+higher order terms\hat{\theta}({\bf z_n}) = \underbrace{\theta_0}_{\text{unbiasedness}} + \sum_{i=1}^n \underbrace{\frac{1}{n} * IF_{\hat{\theta}, F}(z_i)}_{\text{approx. influence of } z_i} + \text{higher order terms}θ^(zn​)=unbiasednessθ0​​​+i=1∑n​approx. influence of zi​n1​∗IFθ^,F​(zi​)​​+higher order terms

An important thing to know is that EF[IFθ^,F(zi)]=0\mathbb{E}_F \left[ IF_{\hat{\theta}, F}(z_i) \right] = 0EF​[IFθ^,F​(zi​)]=0. Therefore, the second term above is going to be approximately zero in large samples, so we have that the mean of θ^(zn)−θ0\hat{\theta}({\bf z_n}) - \theta_0θ^(zn​)−θ0​ is zero (Note this doesn't prove unbiasedness because we are assuming that already.)

This is similar to OLS asymptotics: nβ^−β∗0=1n∑∗i=1n(X′X)−1X′ε+n\*higher order terms\sqrt{n} \hat\beta - \beta*0 = \frac{1}{\sqrt{n}} \sum*{i=1}^n (X'X)^{-1}X'\varepsilon + \sqrt{n} \* \text{higher order terms}n​β^​−β∗0=n​1​∑∗i=1n(X′X)−1X′ε+n​\*higher order terms. Just like in the OLS case, having our Taylor Expansion is helpful because then asymptotics comes easy:

  ⟹  n(θ^(zn)−θ0)=1n∑i=1nIFθ^,F(zi)+n∗higher order terms\implies \sqrt{n}\left( \hat{\theta}({\bf z_n}) - \theta_0 \right) = \frac{1}{\sqrt{n}} \sum_{i=1}^n IF_{\hat{\theta}, F}(z_i) + \sqrt{n} * \text{higher order terms}⟹n​(θ^(zn​)−θ0​)=n​1​i=1∑n​IFθ^,F​(zi​)+n​∗higher order terms

Under some assumptions, the higher order terms go to zero faster than n\sqrt{n}n​ goes to infinity, so the product is approximately zero under large samples.

Therefore, from some central limit theorem:

n(θ^(zn)−θ0)→dN(0,EF[∑i=1nIFθ^,F(zi)′IFθ^,F(zi)])\sqrt{n}\left( \hat{\theta}({\bf z_n}) - \theta_0 \right) \to^d N(0, \mathbb{E}_F \left[ \sum_{i=1}^n IF_{\hat{\theta}, F}(z_i)' IF_{\hat{\theta}, F}(z_i) \right])n​(θ^(zn​)−θ0​)→dN(0,EF​[i=1∑n​IFθ^,F​(zi​)′IFθ^,F​(zi​)])

So, if you know the influence function, then you have large scale asymptotics for free! Then you can just plug in the sample estimates of IFθ^,F(zi)IF_{\hat{\theta}, F}(z_i)IFθ^,F​(zi​) for each ziz_izi​ and you have the variance-covariance matrix for your estimator.

Example 1: Mean

ziz_izi​ is a scalar and our estimator is the population mean of the data, θ^(F)=EF[zi]\hat{\theta}(F) = \mathbb{E}_F\left[ z_i \right]θ^(F)=EF​[zi​].

Our estimator is the sample analogue of: θ^(F)=EF(zi)\hat\theta(F) = E_F(z_i)θ^(F)=EF​(zi​)

Lets think about what happens when we use the contaminated distribution function:

θ^(Fϵ(zi))=EFϵ(zi)[zi]=(1−ϵ)EF[zi]+ϵEδzi[zi]=(1−ϵ)θ^(F)+ϵzi\begin{align*} \hat{\theta}(F_{\epsilon}(z_i)) &= \mathbb{E}_{F_{\epsilon}(z_i)}\left[ z_i \right] \\ &= (1 - \epsilon) \mathbb{E}_F\left[ z_i \right] + \epsilon E_{\delta_{z_i}} \left[ z_i \right] \\ &= (1 - \epsilon) \hat{\theta}(F) + \epsilon z_i \\ \end{align*}θ^(Fϵ​(zi​))​=EFϵ​(zi​)​[zi​]=(1−ϵ)EF​[zi​]+ϵEδzi​​​[zi​]=(1−ϵ)θ^(F)+ϵzi​​

Note that the last equality comes from the fact that we are taking the expectation over the degenerate distribution δzi\delta_{z_i}δzi​​ which has mean ziz_izi​.

Now lets plug that back into the influence function equation

IFθ^,F(zi)=lim⁡ϵ→0θ^(Fϵ(x))−θ^(F)ϵ=(1−ϵ)θ^(F)+ϵzi−θ^(F)ϵ=−ϵθ^(F)+ϵziϵ=zi−EF[zi]\begin{align*} IF_{\hat{\theta}, F}(z_i) &= \lim_{\epsilon \to 0} \frac{\hat{\theta}(F_\epsilon(x)) - \hat{\theta}(F)}{\epsilon} \\ &= \frac{(1 - \epsilon) \hat{\theta}(F) + \epsilon z_i - \hat{\theta}(F)}{\epsilon} \\ &= \frac{-\epsilon \hat{\theta}(F) + \epsilon z_i}{\epsilon} \\ &= z_i - \mathbb{E}_F\left[ z_i \right] \end{align*}IFθ^,F​(zi​)​=ϵ→0lim​ϵθ^(Fϵ​(x))−θ^(F)​=ϵ(1−ϵ)θ^(F)+ϵzi​−θ^(F)​=ϵ−ϵθ^(F)+ϵzi​​=zi​−EF​[zi​]​

This makes intuitive sense. Observations ziz_izi​ are more influential to the sample estimator if they are further away from the population mean.

We can estimate EF[zi]\mathbb{E}_F\left[ z_i \right]EF​[zi​] with the sample mean, and

IF^θ^,F(zi)=zi−zˉ\widehat{IF}_{\hat{\theta}, F}(z_i) = z_i - \bar{z}IFθ^,F​(zi​)=zi​−zˉ

Let θ0\theta_0θ0​ be the true population mean, EF[zi]\mathbb{E}_F\left[ z_i \right]EF​[zi​]. From above and our calculate influence function, we know that

n(zˉ−θ0)∼N(0,1nEF[IFθ^,F′IFθ^,F])=N(0,EF[(zi−zˉ)2]),\sqrt{n}(\bar{z} - \theta_0) \sim N(0, \frac{1}{n}\mathbb{E}_F[ IF_{\hat{\theta}, F}'IF_{\hat{\theta}, F}]) = N(0, \mathbb{E}_F\left[ (z_i-\bar{z})^2\right]),n​(zˉ−θ0​)∼N(0,n1​EF​[IFθ^,F′​IFθ^,F​])=N(0,EF​[(zi​−zˉ)2]),

which we can estimate this variance with 1n∑i=1n(zi−zˉ)2\frac{1}{n} \sum_{i=1}^n (z_i - \bar{z})^2n1​∑i=1n​(zi​−zˉ)2, the sample variance.

Example 2: Regression coefficients

{zi≡(xi,yi)}i=1n\{z_i \equiv (x_i, y_i)\}_{i=1}^n{zi​≡(xi​,yi​)}i=1n​ where xix_ixi​ is a 1×k1\times k1×k vector of covariates and yiy_iyi​ is a scalar outcome. The estimator is

β^(F)=argminβEF[(y−Xβ)′(y−Xβ)] \hat{\beta}(F) = argmin_{\beta} \mathbb{E}_F\left[ (y-X\beta)'(y-X\beta) \right]β^​(F)=argminβ​EF​[(y−Xβ)′(y−Xβ)]

Lets think about what happens when we use the contaminated distribution function:

Which gives first order condition:

(1−ϵ)EF[−2X′y−2X′Xβ^ϵ]+ϵ(2xi′yi−2xi′xiβ^ϵ)=0(1-\epsilon) \mathbb{E}_F[ -2X'y - 2X'X \hat{\beta}_{\epsilon}] + \epsilon(2x_i'y_i - 2 x_i'x_i \hat{\beta}_{\epsilon}) = 0(1−ϵ)EF​[−2X′y−2X′Xβ^​ϵ​]+ϵ(2xi′​yi​−2xi′​xi​β^​ϵ​)=0

Example 3: Treatment Effect

For the last example, we are looking at the average treatment effect among the treated which assumes that counterfactual untreated outcomes are uncorrelated with treatment, i.e. yi(0)⊥Diy_i(0) \perp D_iyi​(0)⊥Di​. Our data is zi=(yi,Di)z_i = (y_i, D_i)zi​=(yi​,Di​) which is iid. Out estimator is the sample analogue of:

τ^(F)=EF[yi∣Di=1]−EF[yi∣Di=0]\hat{\tau}(F) = E_F\left[ y_i \vert D_i = 1\right] - E_F\left[ y_i \vert D_i = 0\right]τ^(F)=EF​[yi​∣Di​=1]−EF​[yi​∣Di​=0]

Note that this is a sum of two estimators from the data: τ1(F)=EF(yi∣Di=1)\tau_1(F) = E_F(y_i \vert D_i = 1)τ1​(F)=EF​(yi​∣Di​=1) and τ0(F)=EF(yi∣Di=0)\tau_0(F) = E_F(y_i \vert D_i = 0)τ0​(F)=EF​(yi​∣Di​=0).

Since influence functions can be expressed as derivatives, we get to use the usual chain rule.

Chain Rule: If θ^=T(θ^1,…,θ^k)\hat{\theta} = T(\hat{\theta}_1, \dots, \hat{\theta}_k)θ^=T(θ^1​,…,θ^k​), then the influence function of θ^\hat{\theta}θ^ is:

In the context of our function θ^(F)=τ1(F)−τ0(F)\hat{\theta}(F) = \tau_1(F) - \tau_0(F)θ^(F)=τ1​(F)−τ0​(F), the influence function is

IFθ^,F(Zi)=1∗IFθ^1,F(Zi)−1∗IFθ^0,F(Zi)IF_{\hat{\theta}, F}(Z_i) = 1 * IF_{\hat{\theta}_1, F}(Z_i) - 1 * IF_{\hat{\theta}_0, F}(Z_i)IFθ^,F​(Zi​)=1∗IFθ^1​,F​(Zi​)−1∗IFθ^0​,F​(Zi​)
