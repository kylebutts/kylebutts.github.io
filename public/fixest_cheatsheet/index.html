Running regressions in R without the hassle.

Many a few individuals have complained about why is running regressions in R so difficult: Robust standard errors? Select covariates with * operator? Absorb fixed effects? Why is it so hard to do things that Stata makes simple?

This is where the package fixest comes in.

Formula Creation

Fixed Effects

                  reghdfe y x, absorb(fe)
                
                  feols(y ~ x | fe, data)

Categorical Variables

                  reghdfe y x i.cat
                
                  feols(y ~ x + i(cat), data)

With Baseline 

                  reghdfe y x ib1.cat
                
                  feols(y ~ x + i(cat, ref = 1), data)

Interact Categoricals

                  reghdfe y x i.cat#i.cat2
                
                  feols(y ~ x + i(cat, i.cat2), data)

Categorical and Continuous

                  reghdfe y x c.cont#i.cat
                
                  feols(y ~ x + i(cat, cont), data)

Macros

                  local vars varlist 
           reghdfe y x `vars'
                
                  vars = c(...) 
           feols(y ~ x + .[vars], data)

Wildcard

                  reghdfe y x age_*
                
                  feols(y ~ x + ..('age_'), data)

Variables v1 ... v4

                  reghdfe y x v1-v4
                
                  feols(y ~ x + v.[1:4], data)

Regex

                  
                
                  feols(y ~ x + ..('regex_exp'), data)

Standard Errors

Robust

                  reghdfe y x, r
                
                  feols(y ~ x, data, 'hetero')

Cluster

                  reghdfe y x, cluster(clust)
                
                  feols(y ~ x, data, vcov = ~clust)

Two-way

                  reghdfe y x, cluster(clust1 clust2)
                
                  feols(y ~ x, data, vcov = ~clust1 + clust2)

Conley Standard Errors

                  
                
                  feols(y ~ x, data, vcov = conley('lat', 'lon', 50))

Post-Estimation

Regression Table 

                  reghdfe ... 
           eststore est1 
           estout ...
                
                  est1 <- feols(...) 
           etable(est1)

Coefficient Plot 

                  reghdfe ... 
           eststore est1 
           estout ...
                
                  est1 <- feols(...) 
           coefplot(est1)

Panel Data

Lag Variables

                  xtset unit time 
           reg y x l1.x l2.x
                
                  panel(df, ~unit + time) 
           feols(y ~ x + l(x, 1:2), df)

Lead Variables 

                  xtset unit time 
           reg y x f1.x f2.x
                
                  panel(df, ~unit + time) 
           feols(y ~ x + l(x, -2:-1), df)

First Difference 

                  xtset unit time 
           reg y x D.x
                
                  panel(df, ~unit + time) 
           feols(y ~ x + d(x), df)

Instrumental Variables

Instrumental Variables

                  ivreghdfe 2sls y x (x1 = z1)
                
                  feols(y ~ x | x1 ~ z1, df)

with Fixed Effects

                  ivreghdfe 2sls y x (x1 = z1), absorb(fe1)
                
                  feols(y ~ x | fe1 | x1 ~ z1, df)
